🧠 My AI Brain API

REST API Documentation — Version 2.0.0

PUBLIC Không cần auth   AUTH Cần API key

🔑 Authenticated endpoints cần header: Authorization: Bearer <API_KEY>
Public endpoints không cần auth — có thể gọi trực tiếp.

💓 Health & Status

GET /api/health PUBLIC ▼

Health check — kiểm tra DB connectivity & memory usage.

{
  "status": "healthy",
  "uptime": 1234,
  "timestamp": "2025-01-15T...",
  "version": "2.0.0",
  "checks": {
    "database": "connected",
    "memory": { "rss": "83MB", "limit": "512MB", "ok": true }
  }
}
GET /api/metrics PUBLIC ▼

Prometheus metrics — monitoring format.

GET /api/scheduler/status PUBLIC ▼

Scheduler job status — xem các cron jobs chạy lần cuối khi nào.

{
  "ok": true,
  "jobs": {
    "pipeline": { "lastRun": "...", "hoursAgo": 2.1, "status": "ok" },
    "memory": { "lastRun": "...", "hoursAgo": 5.3, "status": "ok" },
    "backup": { "lastRun": null, "hoursAgo": null, "status": "never_run" },
    "evo": { "lastRun": "...", "hoursAgo": 15.2, "status": "missed" },
    "graph": { "lastRun": "...", "hoursAgo": 1.0, "status": "ok" }
  }
}
GET /api/sources/health PUBLIC ▼

Source router health — kiểm tra các data sources (YouTube, Reddit, arXiv, GitHub, HN, Tavily).

🤖 AI & RAG

POST /api/ask AUTH ▼

Hỏi đáp với RAG pipeline — tìm kiếm knowledge base + trả lời.

Body:

{
  "query": "What is LSM-Tree?"
}

Response:

{
  "ok": true,
  "answer": "LSM-Tree is a...",
  "source": "knowledge_base",
  "sourcesFormatted": "📚 Sources: ...",
  "resultsCount": 3
}
POST /api/vision/analyze AUTH ▼

Phân tích hình ảnh bằng Gemini Vision. Hỗ trợ cả JSON (base64) và binary upload.

JSON mode:

{
  "image": "base64encoded...",
  "mimeType": "image/png",
  "prompt": "Describe this image"
}

Binary mode:

POST raw image bytes với Content-Type: image/png

POST /api/debate AUTH ▼

Chạy multi-agent debate.

{
  "problem": "Should we use microservices?",
  "quick": false
}

📚 Flashcards (SRS)

GET /api/flashcards AUTH ▼

Lấy random flashcards. Query params: ?limit=20&category=cs

GET /api/flashcards/due AUTH ▼

Lấy flashcards đến hạn ôn tập (spaced repetition).

GET /api/flashcards/stats AUTH ▼

Thống kê flashcards (tổng số, due, mastery...).

POST /api/flashcards AUTH ▼

Tạo flashcard mới.

{
  "question": "What is Raft consensus?",
  "answer": "Raft is a consensus algorithm...",
  "source": "textbook",
  "category": "distributed-systems"
}
POST /api/flashcards/:id/review AUTH ▼

Review flashcard.

{
  "correct": true
}
DELETE /api/flashcards/:id AUTH ▼

Xóa flashcard.
